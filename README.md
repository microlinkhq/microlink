## Install

```bash
$ npm install microlink --global
```

## CLI

```bash
$ microlink --help

  Generates regular expressions that match a set of strings.

  Usage
    $ microlink [-gimuy] string1 string2 string3...

  Examples
    $ microlink foobar foobaz foozap fooza
    $ jq '.keywords' package.json | microlink
```
